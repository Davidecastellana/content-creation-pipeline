# Social Platform Content Specifications

## LinkedIn

### Post Types
| Type | Format | Optimal Length |
|------|--------|----------------|
| Single Post | Text + Image | 150-300 words |
| Document Post | PDF carousel | 10-15 slides |
| Article | Long-form | 1,500-3,000 words |
| Video Post | Native video | Up to 10 minutes |

### Character Limits
- Post text: No hard limit, 150-300 optimal for engagement
- Hashtags: 3-5 recommended

### Best Practices
- Hook in first line (visible without clicking "see more")
- No questions in first line
- Include 1-2 line breaks for readability
- End with question or CTA for comments

---

## Twitter/X

### Post Types
| Type | Format | Requirements |
|------|--------|---------------|
| Single Tweet | Text | 280 chars max |
| Thread | 3-15 tweets | Each tweet standalone-readable |
| Poll | 2-4 options | Question in tweet, options as replies |

### Character Limits
- Single tweet: 280 chars

### Best Practices
- Hook in first 90 chars (most visible)
- Bold statement or contrarian take in tweet 1
- 1-2 relevant hashtags

---

## Instagram

### Post Types
| Type | Format | Requirements |
|------|--------|---------------|
| Feed Post | Image/Video + Caption | 150-500 chars optimal |
| Carousel | Multiple images | Up to 10 slides |
| Reels | Video | 15-60 seconds |
| Story | Image/Video | 15 sec per frame |

### Character Limits
- Caption: 2,200 chars max
- First 2 lines visible before "...more"
- Hashtags: 3-5 in caption, up to 30 in first comment

### Image Specs
| Type | Ratio | Size |
|------|-------|------|
| Feed (preferred) | 4:5 | 1080×1350 |
| Stories/Reels | 9:16 | 1080×1920 |
