# Copy Frameworks for Social Content

## Framework Overview

| Framework | Expansion | Best For | Platform Fit |
|-----------|-----------|----------|--------------|
| AIDA | Attention, Interest, Desire, Action | Cold audiences, awareness | LinkedIn, Twitter, Instagram |
| PAS | Problem, Agitate, Solution | Pain-point products | LinkedIn, Twitter |
| BAB | Before, After, Bridge | Transformation offers | Instagram, Twitter |
| 4P | Promise, Picture, Proof, Push | High-ticket, B2B | LinkedIn |
| FAB | Features, Advantages, Benefits | Technical products | LinkedIn, Twitter |

---

## AIDA Framework

**Steps:** Attention > Interest > Desire > Action

### LinkedIn Template
- Hook in first line (value statement)
- Interest: supporting detail or insight
- Desire: benefit-focused emotional appeal
- Action: clear CTA

### Twitter Template
- Attention: Bold statement or stat in first 90 chars
- Interest: detail in remaining chars
- Desire: benefit statement
- Action: engagement CTA

### Instagram Template
- Hook: First 2 lines before "...more"
- Interest: main content
- Desire: benefit or insight
- Action: question or save CTA

---

## PAS Framework

**Steps:** Problem > Agitate > Solution

### LinkedIn Template
- Problem: State the pain point professionally
- Agitate: Amplify the impact on KPIs or goals
- Solution: Your offering as natural relief

### Twitter Template
- Problem: Single tweet, punchy
- Agitate: Thread continuation
- Solution: Brand as solution

---

## BAB Framework

**Steps:** Before > After > Bridge

### LinkedIn Template
- Before: Current professional challenge
- After: Desired state with improved metrics
- Bridge: How your solution enables the transformation

### Instagram Template
- Before: Pain point or current state
- After: Desired state
- Bridge: Product/service

---

## 4P Framework

**Steps:** Promise > Picture > Proof > Push

### LinkedIn Template
- Promise: Compelling headline about outcome
- Picture: Vivid visualization of benefit
- Proof: Data point or testimonial
- Push: Clear CTA

---

## FAB Framework

**Steps:** Features > Advantages > Benefits

### LinkedIn Template
- Feature: What it is
- Advantage: Why it's better than alternatives
- Benefit: What the user gains

---

## Copy Quality Rules

1. **Never exceed character limits** - truncated copy destroys engagement
2. **Always show character count** in parentheses
3. **Hook in first 3 words** - first line must stop the scroll
4. **Active voice only** - no passive constructions
5. **Every CTA must have action verb + benefit**
6. **Numbers work** - include specific stats where relevant
