# Content Creation Pipeline

## Overview
This pipeline generates social media content (LinkedIn, Twitter, Instagram) through a 4-stage workflow with competitive research, brief collection, content creation, and output formatting.

## Stage Order
001-RESEARCH → 002-INTAKE → 003-CREATE → 004-OUTPUT

## Stage Purposes
| Stage | Type | Purpose |
|-------|------|---------|
| 001-RESEARCH | HYBRID | Competitive analysis, trend research, content intelligence |
| 002-INTAKE | INTERACTIVE | Collect brief, topic, requirements from user |
| 003-CREATE | AUTO | Generate content using copywriting and creative skills |
| 004-OUTPUT | AUTO | Format and package content for each platform |

## Skills Mapping
| Stage | Skills |
|-------|--------|
| 001-RESEARCH | competitor-analysis, trend-research |
| 002-INTAKE | brief-collection, topic-definition |
| 003-CREATE | copywriting, social-content |
| 004-OUTPUT | format-adapter, platform-optimization |

## Inter-Pipeline Inputs
Reads from: Competitive research data, industry trends
